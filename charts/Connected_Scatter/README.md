# 连线散点图 Connected Scatterplot

图表分类: 时间趋势类 TIME
感知排名: ⭐⭐⭐⭐☆

## 简介

连线散点图在散点图的基础上，用线段连接各点展示数据的演变过程或轨迹。它通过连接线和标记点的组合，使用户可以直观观察：

- 数据点之间的演变过程和路径
- 因果关系或时间序列的发展趋势
- 多个变量间的相关性和变化规律

相比散点图，连线散点图更适合展示有序的演变过程；相比折线图，连线散点图更适合展示非时间序列的关系演变。


## 数据格式

需要包含以下字段：

- x 列（数值）：X 轴数据，通常为某个指标或维度
- y 列（数值）：Y 轴数据，通常为另一个指标或维度
- order 列（可选）：排序列，用于确定连接顺序（时间、序列等）
- size 列（可选）：标记大小列，用于表示第三个维度

典型数据结构：

| 年份 | 销售额 | 利润 | 增长率 |
|------|--------|------|--------|
| 2018 | 100 | 20 | 5% |
| 2019 | 150 | 35 | 8% |
| 2020 | 120 | 28 | 6% |
| 2021 | 200 | 50 | 12% |


## 适用场景

连线散点图适用于以下场景：

- 产品演变过程：展示产品从推出到成熟的销售额和利润变化
- 公司发展轨迹：展示公司在不同时期的收入和成本关系
- 学生成长过程：展示学生在不同阶段的成绩和排名变化
- 市场演变分析：展示市场份额和价格的演变关系


## 典型场景

- 销售与利润演变：展示 5 年间销售额与利润的相关性变化
- 股价与成交量：展示股票价格与成交量的演变轨迹
- 用户增长与留存：展示新增用户与留存率的演变过程
- 成本与效率：展示生产成本与生产效率的演变关系


## 使用建议

- 数据点建议 ≥ 3 个，以展示明显的演变过程
- 使用 order 列确保连接顺序正确（特别是非时间序列数据）
- 可选 size 列表示第三个维度，增加信息密度
- 标记点大小会根据 size 列自动缩放（如果提供）


## 注意事项

- 缺失值会被自动删除，可能导致连接线中断
- x 和 y 列必须可转换为数值类型
- order 列用于确定连接顺序，如不提供则按数据行顺序连接
- size 列会自动缩放到 6-14px 范围内


## 与相似图表的区别

### 连线散点图 vs 散点图

- 连线散点图：点之间有连接线，强调演变过程和路径
- 散点图：仅显示点，强调点的分布和相关性

### 连线散点图 vs 折线图

- 连线散点图：适合非时间序列的关系演变，点的大小可表示第三维度
- 折线图：适合时间序列数据，强调时间趋势

### 连线散点图 vs 气泡图

- 连线散点图：强调演变过程，点之间有连接线
- 气泡图：强调三维关系，点的大小表示第三维度，无连接线
